# R Statistical Analysis Using House Prices Dataset
This repository contains beginner to intermediate statistical analysis in R using the House Prices dataset. It is created as part of my MBA in Business Analytics learning journey to understand data visualization, normality testing, hypothesis testing, and business data interpretation.

## Repository Contents

### Dataset Used
- **houseprices.xlsx**  
A sample housing dataset containing variables such as Price, Square Feet, Offers, Brick type, and Neighborhood.

### R Files Covered

### 1. Exploratory Data Analysis (EDA)
Includes:
- Scatter plots
- Correlation analysis
- Boxplots
- Histograms
- Median calculation
- Frequency tables
- Bar plots

Key Concepts:
- Relationship between house price and size
- Price comparison by neighborhood
- Price comparison by brick type

### 2. Normality Testing
Includes:
- Histogram with density curve
- Boxplot
- Q-Q Plot
- Shapiro-Wilk Test
- Lilliefors Test
- Anderson-Darling Test

Purpose:
To check whether house prices follow normal distribution.

### 3. Random Normal Data Practice
Includes:
- Generating random normal values using `rnorm()`
- Histogram
- Boxplot
- Q-Q Plot

Purpose:
To understand how normally distributed data looks in practice.

### 4. One Sample T-Test
Used to test whether average house price is significantly different from a benchmark value.
Example:
- Mean Price = 130000

### 5. Independent Two Sample T-Test
Used to compare average house prices between:
- Brick houses vs Non-brick houses

### 6. Levene’s Test
Used to test equality of variances before performing t-test.
